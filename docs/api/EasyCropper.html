Class: EasyCropper

EasyCropper()

A custom element to provide functionality for cropping images.

Constructor

new EasyCropper()

Properties:
Name Type Description
aspectRatio string The aspect ratio to be cropped. Can be either a value or an equation like 16:9.
format ImageFormat The format of the output image to be generated.
maxZoom number The maximum zoom level allowed. Default is 5.
padding number The padding around the view-finder in pixels. Default is 64px.
noPixels boolean Whether to display interpolated pixels when the image is zoomed in or not.
quality number A number between 0 and 1 indicating the image quality if the format is "jpeg" or "webp".
src string The source URL of the image to be cropped.
